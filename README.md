# Welcome to Forge Code 1

**Forge Code 1** is a simple, expressive programming language designed to be easy to learn and compile directly from `.fc1` files into Rust programs. 

It combines the clarity of modern scripting languages with the power of compiled Rust, making it perfect for learning, experimenting, and building small projects.

## Key Features

- **Direct execution from `.fc1` files** (no `.rs` files needed)
- **Simple syntax:** variables, print statements, and math
- **Fast compilation** leveraging Rust
- Beginner-friendly, easy to read and extend
